### Dependencies:

      1.cmake   
  
      2.openCV  
To install dependecies:

      sudo apt-get install libopencv-dev

      sudo apt-get install cmake 
### Build instructions:
To build, execute the following commands from the project root directory:

    mkdir build

    cd build

    cmake ../

    make
### To Run:
From the build directory:

    ./ComputationalGeometry
